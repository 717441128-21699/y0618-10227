## 1. 产品概述

显微镜图像拼接与细胞计数分析平台，面向生物实验室研究人员。系统将同一样本在不同视野拍摄的多张显微图像，通过自动特征点匹配拼接为无缝大视野全景图并均衡重叠区亮度；随后对全景图中的细胞/颗粒/菌落进行自动分割、计数与形态测量，支持按大小、圆度等参数过滤目标，并提供人工修正、多实验组批量统计对比，最终导出带标注图像与统计数据表格用于论文发表。

- 目标用户：生物/医学实验室研究人员、研究生，需要快速完成"拍照→拼接→计数→统计→出图"全流程
- 核心价值：把传统需 ImageJ 多步手动操作的专业流程，整合为一个可视化、可追溯、可对比的一体化工作台

## 2. 核心功能

### 2.2 功能模块
1. **实验工作台**：实验组与样本管理、工作流引导、全局进度概览
2. **图像拼接工作室**：多图批量上传、缩略图排序、特征点自动匹配、全景拼接、重叠区亮度均衡
3. **细胞计数工作室**：自动分割计数、形态参数阈值过滤、画布上手绘增删标记修正
4. **测量与统计**：单目标形态测量（面积/周长/长短轴）、整批样本参数分布直方图、组内汇总
5. **对比与导出**：多实验组计数结果柱状对比、导出带标注 PNG、导出统计 CSV

## 3. 核心流程

用户在实验工作台新建实验组并导入多视野图片 → 进入图像拼接工作室上传图片并排序 → 系统自动提取特征点并匹配，估算图间位移，执行多频带融合拼接并均衡重叠区亮度，生成全景图 → 进入细胞计数工作室，选择目标类型并运行自动分割，调整形态过滤阈值筛选有效目标 → 在画布上人工增删标记修正遗漏 → 进入测量与统计查看目标形态测量值与参数分布 → 进入对比与导出，将多个实验组计数结果汇总对比，导出带标注图像与统计表格。

## 4. 用户界面设计

### 4.1 设计风格
- **整体定位**："数字暗房 / 精密仪器台" —— 深色暗房氛围搭配荧光级高亮数据可视化，专业、精密、可信赖
- **主色与辅色**：背景深炭蓝（#0a0e14）+ 面板（#12161f）；主强调色为荧光青（#2dd4bf，呼应荧光显微）；次强调色为琥珀金（#fbbf24，用于测量与高亮）；危险色珊瑚红（#f87171）
- **按钮风格**：紧凑矩形、细描边、低圆角（2-4px），主按钮青色微发光，工具按钮图标式
- **字体**：标题用 IBM Plex Sans（科学仪器气质），数据/测量值/坐标用 IBM Plex Mono（等宽，对齐表格数值）
- **布局风格**：工作台式，左侧固定导航栏 + 顶部工具条 + 中央画布 + 右侧检查器面板，密集但留呼吸空间
- **图标风格**：线性细描边图标，2px 描边，与精密仪器风格一致

### 4.3 响应式
桌面优先（实验室均为桌面工作站），最低 1280px 宽工作台布局；窄屏时右侧检查器折叠为抽屉，画布自适应。不做移动端触控优化。
